# agentspace skills

## Installation
Send the following to your agent,
```
I'd like you to set up https://agentspace.so, the file sharing for AI agents.
Install as a skill if I have npm: npx skills add agentspace-so/skills --skill agentspace -g
If not, do this instead: curl -fsSL https://agentspace.so/install.sh | bash
```

## Usage
Ask the following to your agent,
```
Share the current folder with me.
```
or
```
Share your recent generated files with me.
```
